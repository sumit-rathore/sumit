This test tests the rate at which the board flashes n words.  It begins by erasing the memory.  It then prints an ilog message.  It then flashes the board.  It then prints another ilog message.  The test can flash any number of words per page, whcih is modified by changing the define at the top of flash_test.c, additionally the total number of pages can also be modified by changing the define at the top of flash_test.c
